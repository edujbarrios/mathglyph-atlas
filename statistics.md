# Statistics and Probability

## Probability and Statistics Symbols

| Name | Rendered Symbol | LaTeX Code |
|---|---|---|
| Probability | P(A) | `P(A)` |
| Conditional Probability | P(A\|B) | `P(A\mid B)` |
| Independence | ⫫ | `\perp\!\!\!\perp` |
| Expected Value | E[X] | `\mathbb{E}[X]` |
| Variance | Var(X) | `\mathrm{Var}(X)` |
| Standard Deviation | σ | `\sigma` |
| Covariance | Cov(X,Y) | `\mathrm{Cov}(X,Y)` |
| Correlation | ρ | `\rho` |
| Mean | μ | `\mu` |
| Sample Mean | x̄ | `\bar{x}` |
| Median | x̃ | `\tilde{x}` |
| Maximum Likelihood Estimate | θ̂ | `\hat{\theta}` |
| Argmax | argmax | `\arg\max_{\theta}` |
| Argmin | argmin | `\arg\min_{\theta}` |
| Normal Distribution | N(μ,σ²) | `\mathcal{N}(\mu,\sigma^2)` |
| Bernoulli Distribution | Bern(p) | `\mathrm{Bernoulli}(p)` |
| Binomial Distribution | Bin(n,p) | `\mathrm{Binomial}(n,p)` |
| Poisson Distribution | Pois(λ) | `\mathrm{Poisson}(\lambda)` |
| Exponential Distribution | Exp(λ) | `\mathrm{Exp}(\lambda)` |
| Uniform Distribution | U(a,b) | `\mathcal{U}(a,b)` |
| Cumulative Distribution Function | F(x) | `F_X(x)` |
| Probability Density Function | f(x) | `f_X(x)` |
| Likelihood | L(θ) | `\mathcal{L}(\theta)` |
| Log-Likelihood | ℓ(θ) | `\ell(\theta)` |
| Bayes' Rule | P(A\|B) | `P(A\mid B)=\frac{P(B\mid A)P(A)}{P(B)}` |
| Law of Total Probability | Σ | `P(A)=\sum_i P(A\mid B_i)P(B_i)` |
