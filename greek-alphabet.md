# Greek Alphabet

## Complete Greek Alphabet (Uppercase and Lowercase)

| Letter Name | Rendered Symbol | LaTeX Code |
|---|---|---|
| Alpha | Α / α | `\Alpha`, `\alpha` |
| Beta | Β / β | `\Beta`, `\beta` |
| Gamma | Γ / γ | `\Gamma`, `\gamma` |
| Delta | Δ / δ | `\Delta`, `\delta` |
| Epsilon | Ε / ε (ϵ) | `\Epsilon`, `\epsilon`, `\varepsilon` |
| Zeta | Ζ / ζ | `\Zeta`, `\zeta` |
| Eta | Η / η | `\Eta`, `\eta` |
| Theta | Θ / θ (ϑ) | `\Theta`, `\theta`, `\vartheta` |
| Iota | Ι / ι | `\Iota`, `\iota` |
| Kappa | Κ / κ (ϰ) | `\Kappa`, `\kappa`, `\varkappa` |
| Lambda | Λ / λ | `\Lambda`, `\lambda` |
| Mu | Μ / μ | `\Mu`, `\mu` |
| Nu | Ν / ν | `\Nu`, `\nu` |
| Xi | Ξ / ξ | `\Xi`, `\xi` |
| Omicron | Ο / ο | `\Omicron`, `o` |
| Pi | Π / π (ϖ) | `\Pi`, `\pi`, `\varpi` |
| Rho | Ρ / ρ (ϱ) | `\Rho`, `\rho`, `\varrho` |
| Sigma | Σ / σ (ς) | `\Sigma`, `\sigma`, `\varsigma` |
| Tau | Τ / τ | `\Tau`, `\tau` |
| Upsilon | Υ / υ | `\Upsilon`, `\upsilon` |
| Phi | Φ / φ (ϕ) | `\Phi`, `\phi`, `\varphi` |
| Chi | Χ / χ | `\Chi`, `\chi` |
| Psi | Ψ / ψ | `\Psi`, `\psi` |
| Omega | Ω / ω | `\Omega`, `\omega` |

## Common Mathematical Usage

| Letter Name | Rendered Symbol | LaTeX Code |
|---|---|---|
| Angle | θ | `\theta` |
| Eigenvalue | λ | `\lambda` |
| Learning Rate / Small Constant | η | `\eta` |
| Mean (sometimes) | μ | `\mu` |
| Standard Deviation | σ | `\sigma` |
| Variance / Potential Function | φ | `\phi` |
| Big-Oh Reference Symbol | Ω | `\Omega` |
| Error Term | ε | `\epsilon` |
| Summation Index (sometimes) | κ | `\kappa` |
| Probability Density (sometimes) | ρ | `\rho` |
