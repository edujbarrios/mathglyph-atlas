# Greek Alphabet

## Complete Greek Alphabet (Uppercase and Lowercase)

| Letter Name | Rendered Symbol | LaTeX Code |
|---|---|---|
| Alpha | Α / α | `A`, `\alpha` |
| Beta | Β / β | `B`, `\beta` |
| Gamma | Γ / γ | `\Gamma`, `\gamma` |
| Delta | Δ / δ | `\Delta`, `\delta` |
| Epsilon | Ε / ε (ϵ) | `E`, `\epsilon`, `\varepsilon` |
| Zeta | Ζ / ζ | `Z`, `\zeta` |
| Eta | Η / η | `H`, `\eta` |
| Theta | Θ / θ (ϑ) | `\Theta`, `\theta`, `\vartheta` |
| Iota | Ι / ι | `I`, `\iota` |
| Kappa | Κ / κ (ϰ) | `K`, `\kappa`, `\varkappa` |
| Lambda | Λ / λ | `\Lambda`, `\lambda` |
| Mu | Μ / μ | `M`, `\mu` |
| Nu | Ν / ν | `N`, `\nu` |
| Xi | Ξ / ξ | `\Xi`, `\xi` |
| Omicron | Ο / ο | `O`, `o` |
| Pi | Π / π (ϖ) | `\Pi`, `\pi`, `\varpi` |
| Rho | Ρ / ρ (ϱ) | `P`, `\rho`, `\varrho` |
| Sigma | Σ / σ (ς) | `\Sigma`, `\sigma`, `\varsigma` |
| Tau | Τ / τ | `T`, `\tau` |
| Upsilon | Υ / υ | `\Upsilon`, `\upsilon` |
| Phi | Φ / φ (ϕ) | `\Phi`, `\phi`, `\varphi` |
| Chi | Χ / χ | `X`, `\chi` |
| Psi | Ψ / ψ | `\Psi`, `\psi` |
| Omega | Ω / ω | `\Omega`, `\omega` |
