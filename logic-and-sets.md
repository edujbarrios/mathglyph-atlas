# Logic and Set Theory

## Set Theory Symbols

| Name | Rendered Symbol | LaTeX Code |
|---|---|---|
| Empty Set | ∅ | `\emptyset` |
| Element Of | ∈ | `\in` |
| Not Element Of | ∉ | `\notin` |
| Subset | ⊂ | `\subset` |
| Subset or Equal | ⊆ | `\subseteq` |
| Proper Superset | ⊃ | `\supset` |
| Superset or Equal | ⊇ | `\supseteq` |
| Union | ∪ | `\cup` |
| Intersection | ∩ | `\cap` |
| Set Difference | \ | `\setminus` |
| Symmetric Difference | △ | `\triangle` |
| Cartesian Product | × | `A \times B` |
| Power Set | P(A) | `\mathcal{P}(A)` |
| Cardinality | |A| | `\lvert A \rvert` |
| Natural Numbers | ℕ | `\mathbb{N}` |
| Integers | ℤ | `\mathbb{Z}` |
| Rational Numbers | ℚ | `\mathbb{Q}` |
| Real Numbers | ℝ | `\mathbb{R}` |
| Complex Numbers | ℂ | `\mathbb{C}` |
