# Number Theory

## Number Theory Symbols

| Name | Rendered Symbol | LaTeX Code |
|---|---|---|
| Divides | a\|b | `a \mid b` |
| Does Not Divide | a∤b | `a \nmid b` |
| Congruent Modulo n | a≡b (mod n) | `a \equiv b \pmod{n}` |
| Not Congruent Modulo n | a≢b (mod n) | `a \not\equiv b \pmod{n}` |
| Greatest Common Divisor | gcd(a,b) | `\gcd(a,b)` |
| Least Common Multiple | lcm(a,b) | `\mathrm{lcm}(a,b)` |
| Prime Number | p | `p` |
| Composite Number | n | `n` |
| Euler's Totient | φ(n) | `\varphi(n)` |
| Möbius Function | μ(n) | `\mu(n)` |
| Divisor Function | σ(n) | `\sigma(n)` |
| Prime Counting Function | π(x) | `\pi(x)` |
| Legendre Symbol | (a/p) | `\left(\frac{a}{p}\right)` |
| Jacobi Symbol | (a/n) | `\left(\frac{a}{n}\right)` |
| Modular Inverse | a⁻¹ mod n | `a^{-1} \bmod n` |
| Chinese Remainder System | x≡aᵢ mod nᵢ | `x \equiv a_i \pmod{n_i}` |
