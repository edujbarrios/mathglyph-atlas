# Calculus

## Differential and Integral Notation

| Name | Rendered Symbol | LaTeX Code |
|---|---|---|
| Limit | lim | `\lim_{x \to a} f(x)` |
| Derivative | d/dx | `\frac{d}{dx}` |
| Second Derivative | d²/dx² | `\frac{d^2}{dx^2}` |
| Partial Derivative | ∂ | `\frac{\partial f}{\partial x}` |
| Mixed Partial Derivative | ∂²/(∂x∂y) | `\frac{\partial^2 f}{\partial x\partial y}` |
| Gradient | ∇f | `\nabla f` |
| Divergence | ∇·F | `\nabla \cdot \mathbf{F}` |
| Curl | ∇×F | `\nabla \times \mathbf{F}` |
| Laplacian | ∇² | `\nabla^2` |
| Differential | dx | `dx` |
| Definite Integral | ∫ | `\int_{a}^{b} f(x)\,dx` |
| Indefinite Integral | ∫ | `\int f(x)\,dx` |
| Double Integral | ∬ | `\iint_{D} f(x,y)\,dA` |
| Triple Integral | ∭ | `\iiint_{V} f(x,y,z)\,dV` |
| Contour Integral | ∮ | `\oint_{C} f(z)\,dz` |
| Line Integral | ∫C | `\int_{C} \mathbf{F}\cdot d\mathbf{r}` |
| Surface Integral | ∬S | `\iint_{S} \mathbf{F}\cdot d\mathbf{S}` |
| Jacobian | J | `\frac{\partial(x,y)}{\partial(u,v)}` |
| Taylor Series | Σ | `\sum_{n=0}^{\infty} \frac{f^{(n)}(a)}{n!}(x-a)^n` |
| Maclaurin Series | Σ | `\sum_{n=0}^{\infty} \frac{f^{(n)}(0)}{n!}x^n` |
| Fundamental Theorem of Calculus | ∫ and d/dx | `\frac{d}{dx}\int_{a}^{x} f(t)\,dt = f(x)` |
| Big O Remainder | O(h²) | `O(h^2)` |
