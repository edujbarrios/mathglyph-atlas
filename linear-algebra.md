# Linear Algebra

## Matrix and Vector Notation

| Name | Rendered Symbol | LaTeX Code |
|---|---|---|
| Vector | **v** | `\mathbf{v}` |
| Matrix | A | `\mathbf{A}` |
| Zero Vector | 0 | `\mathbf{0}` |
| Identity Matrix | I | `\mathbf{I}` |
| Transpose | Aᵀ | `A^\top` |
| Inverse | A⁻¹ | `A^{-1}` |
| Pseudoinverse | A⁺ | `A^{+}` |
| Determinant | det(A) | `\det(A)` |
| Trace | tr(A) | `\mathrm{tr}(A)` |
| Rank | rank(A) | `\mathrm{rank}(A)` |
| Null Space | Null(A) | `\mathrm{Null}(A)` |
| Column Space | Col(A) | `\mathrm{Col}(A)` |
| Row Space | Row(A) | `\mathrm{Row}(A)` |
| Span | span{vᵢ} | `\mathrm{span}\{v_i\}` |
| Basis | B | `\mathcal{B}` |
| Inner Product | ⟨x,y⟩ | `\langle x,y \rangle` |
| Dot Product | x·y | `x \cdot y` |
| Cross Product | x×y | `x \times y` |
| Norm | ‖x‖₂ | `\lVert x \rVert_2` |
| Frobenius Norm | ‖A‖F | `\lVert A \rVert_F` |
| Eigenvalue Equation | Av=λv | `A\mathbf{v}=\lambda\mathbf{v}` |
| Eigenvalues | λᵢ | `\lambda_i` |
| Eigenvectors | vᵢ | `\mathbf{v}_i` |
| Diagonal Matrix | diag(d) | `\mathrm{diag}(d_1,\dots,d_n)` |
| Matrix Multiplication | AB | `AB` |
| Kronecker Product | A⊗B | `A \otimes B` |
| Hadamard Product | A∘B | `A \circ B` |
